 16) Qu'est ce que Docker ? A quoi cela sert-il ?

 Docker sert à conteneuriser des applications, cela permet de lancer l'application dans le même environnement peu
 importe la machine sur lequel il sera lancé

 17) Qu'est ce que la CI et la CD ? Pourquoi est-ce utile ?

 La CI/CD permet d'accélerer les tests dans le temps de développement afin d'être sûr du code qui va partir en production
En effet on peut imaginer une pipeline qui va executer des tests chaque fois qu'on push du code sur une branche de dev
et ensuite une autre pipeline qui va deployer ce code quand il sera push sur une branche main
